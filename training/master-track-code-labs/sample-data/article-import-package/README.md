# Nexcent Article import package source

This directory is the auditable source of `../nexcent-article-import.zip`.

- `manifest.json` selects the versioned `NXC_ARTICLE_V1` handler.
- `articles.xlsx` contains `Articles`, `Assets`, `Taxonomy`, and `Instructions`.
- `assets/` contains the binaries referenced by the workbook's `Assets` sheet.

Rebuild the fixture from this directory and keep the entry names unchanged.
